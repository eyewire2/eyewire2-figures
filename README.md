# Content

This repository collects the code that is necessary to reproduce the figures from the Eyewire II resource paper.

# Setup

## Python

Run the following command to setup the python environment:
```
uv sync
```


## Data

Download the data from: https://huggingface.co/datasets/eulerlab/eyewire2-data/tree/main and copy it to [data](./data).

# Notebooks

Run the notebooks in [notebooks](./notebooks) to create figure panels from the paper.
